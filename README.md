# 𝕜𝕚𝕝𝕝𝕤𝕙𝕖𝕞𝕒𝕣 🔪

a simple two player game made with socket.io & p5.js 

just for fun!

### 𝔓𝔯𝔢𝔯𝔢𝔮𝔲𝔦𝔰𝔦𝔱𝔢𝔰

Commands to run project

```

  # cd ks-backend
  $ bundle install
  $ rails db:seed (optional)
  $ rails s
  
  # cd ks-frontend, start socket.io server
  $ npm install
  $ node server.js
  
  # start server
  $ http-server

```
